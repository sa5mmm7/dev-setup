# Step 4 - Python

This step mirrors `mac-onboarding/scripts/04-python/README.md`.

Windows/WSL2 differences:

- Installs pyenv build dependencies with `apt`
- Installs pyenv with the upstream curl installer
- Updates your detected WSL2 shell profile, usually `~/.bashrc` or `~/.zshrc`
- Installs Python versions under the WSL2 Linux home directory
